# AI University Helpdesk

AI powered multilingual chatbot for universities.

## Features
- AI chatbot
- Voice input
- Voice output
- Multilingual support
- React frontend
- FastAPI backend

## Tech Stack
Frontend: React  
Backend: FastAPI  
AI Model: Llama3 (Groq API)  
Speech: ElevenLabs

## How to run

Backend:
uvicorn main:app --reload

Frontend:
npm start
